# ScriptFlow Voiceover Script Generator

Runnable local build for the ScriptFlow voiceover script generation UI.

## Run

Open `index.html` directly in a browser. No install step or dev server is required.

## Implemented

- Dashboard with ScriptFlow header, upload zone, script options, stats, and recent projects.
- Video upload or drag-and-drop flow with simulated AI processing.
- Processing screen with video metadata, waveform animation, progress, ETA, and checklist.
- Script editor with playback simulation, timestamped editable segments, local save, copy, share, and export modal.
- Export choices for TXT, DOCX, PDF, SRT, JSON, and CSV.
- Settings, project board, notifications, search, theme toggle, and responsive mobile layout.

The original UI/UX images, PDFs, React export, JSX source, and ZIP archive are preserved as design/reference assets.
