This is a fully interactive ScriptFlow voiceover script generation app built from your complete UI/UX design specs. Here's what's implemented across all screens:

Screen 1 — Upload Dashboard (Home):
Hero stats cards (Total Scripts, Processing, This Month, Avg Quality), prominent drag-and-drop upload area with gradient icon, file format badges, recent projects grid with colored thumbnails, status badges, and F-pattern layout with value proposition at top.

Screen 2 — AI Processing:
Two-column layout (30/70 split per spec) with video preview on the left showing file details, and real-time AI processing on the right with animated progress bar, rotating brain icon, 7-step checklist with live status indicators (complete/active/pending), percentage counter, and ETA. Auto-transitions to editor at 100%.

Screen 3 — Script Editor:
Two-panel layout with video player (left, 40%) including playback controls, timeline scrubber, play/pause/skip, time display, and color-coded scene detection bar. Right panel has timestamped script segments with inline editing, word counts, scene descriptions, copy buttons, and a "Playing" indicator that syncs with video time. Top toolbar includes formatting tools (Bold/Italic/Underline, Undo/Redo) and Export/Share buttons.

Screen 4 — Settings:
Left sidebar with 8 organized categories (Appearance, Voice & Language, Export, Accessibility, Notifications, Account, Security, Billing). Appearance tab includes Light/Dark/System theme selector with icons, font size dropdown, and reduced animations toggle. Voice tab has language and tone selectors.

Export Modal:
6 format options (TXT, DOCX, PDF, SRT, JSON, CSV) in a 3-column grid with icons, toggle switches for timestamps and scene descriptions, and prominent export CTA button.

Global Features:
Collapsible sidebar navigation with animation, dark mode toggle in top bar, notification dropdown with 4 notification types (success/progress/info/warning) with read/unread states, search bar, user avatar, toast notifications, and seamless screen transitions — all following the deep blue (#2563EB) / teal (#0891B2) color system with WCAG AA contrast compliance.
